## 方案概述

基于现有前端架构（React + SSE流式），添加一个可折叠的实时控制台面板，用于显示研究全流程的详细输出。

## 实现计划

### 1. 新增类型定义 (types/index.ts)
- 添加 `ConsoleLog` 类型，支持多种日志级别（info, success, warning, error, tool_call, tool_result）
- 添加 `StreamEvent` 类型扩展

### 2. 扩展 Research Store (stores/researchStore.ts)
- 添加 `consoleLogs` 状态数组
- 添加 `addConsoleLog`, `clearConsoleLogs`, `toggleConsole` 等方法
- 添加 `isConsoleOpen` 控制面板展开/折叠

### 3. 创建 ConsolePanel 组件 (components/ConsolePanel.tsx)
**功能特性：**
- 可折叠/展开的面板设计
- 实时显示日志流（带时间戳、级别颜色区分）
- 支持工具调用和结果的格式化显示
- 一键清空日志
- 自动滚动到底部
- 支持手动暂停/恢复自动滚动

**日志类型显示：**
- 🔵 INFO: 一般信息（白色）
- 🟢 SUCCESS: 成功状态（绿色）
- 🟡 WARNING: 警告（黄色）
- 🔴 ERROR: 错误（红色）
- 🔧 TOOL_CALL: 工具调用（蓝色）
- 📊 TOOL_RESULT: 工具结果（紫色）

### 4. 修改 ResearchChat 组件
- 集成 ConsolePanel 组件
- 在 SSE 事件处理中同步添加控制台日志
- status 事件 → INFO 日志
- tool_call 事件 → TOOL_CALL 日志
- tool_result 事件 → TOOL_RESULT 日志
- complete 事件 → SUCCESS 日志
- error 事件 → ERROR 日志

## 技术实现细节

### 前端状态流
```
SSE Event → ResearchStore.updateStatus()
         ↓
    ResearchStore.addConsoleLog()
         ↓
    ConsolePanel 自动更新
```

### 关键代码示例
```typescript
// ConsolePanel.tsx
const ConsolePanel = () => {
  const { consoleLogs, isConsoleOpen, toggleConsole } = useResearchStore()
  const scrollRef = useRef<HTMLDivElement>(null)
  
  useEffect(() => {
    if (scrollRef.current && isAutoScroll) {
      scrollRef.current.scrollTop = scrollRef.current.scrollHeight
    }
  }, [consoleLogs])
  
  return (
    <div className={`console-panel ${isConsoleOpen ? 'open' : 'collapsed'}`}>
      <div className="console-header" onClick={toggleConsole}>
        <span>▶ 实时控制台 ({consoleLogs.length})</span>
        <div className="console-actions">
          <button onClick={clearLogs}>清空</button>
          <button onClick={toggleAutoScroll}>
            {isAutoScroll ? '⏸' : '▶'}
          </button>
        </div>
      </div>
      {isConsoleOpen && (
        <div className="console-content" ref={scrollRef}>
          {consoleLogs.map((log, i) => (
            <ConsoleLogItem key={i} log={log} />
          ))}
        </div>
      )}
    </div>
  )
}
```
