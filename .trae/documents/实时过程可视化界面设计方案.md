## 设计方案概述

基于现有代码架构，设计一个三栏式实时过程可视化界面：

### 布局结构
```
┌─────────────────┬─────────────────────┬─────────────────┐
│   左侧：研究对话   │   中间：过程可视化    │   右侧：最终报告  │
│  ResearchChat   │  ProcessVisualizer  │  ReportViewer   │
│                 │                     │                 │
│  - 输入框        │  ┌───────────────┐  │  - 结构化报告    │
│  - 消息历史      │  │  规划阶段      │  │  - 参考文献     │
│  - 示例问题      │  │  PlanningPanel│  │                 │
│                 │  └───────────────┘  │                 │
│                 │  ┌───────────────┐  │                 │
│                 │  │  搜索阶段      │  │                 │
│                 │  │  SearchPanel  │  │                 │
│                 │  │  (可折叠列表)   │  │                 │
│                 │  └───────────────┘  │                 │
│                 │  ┌───────────────┐  │                 │
│                 │  │  分析阶段      │  │                 │
│                 │  │ AnalysisPanel │  │                 │
│                 │  └───────────────┘  │                 │
└─────────────────┴─────────────────────┴─────────────────┘
```

### 核心组件

1. **ProcessVisualizer** (新建)
   - 中央面板，包含三个阶段子面板
   - 通过 Tabs 或垂直布局组织

2. **PlanningPanel** (规划阶段)
   - 显示研究计划和搜索查询
   - 实时展示模型 streaming 响应
   - 卡片式布局带时间戳

3. **SearchPanel** (搜索阶段)
   - 可折叠手风琴列表 (Accordion)
   - 每轮搜索独立分组
   - 显示搜索问题、结果数量、摘要

4. **AnalysisPanel** (分析阶段)
   - 实时分析过程展示
   - 时间线或步骤式布局
   - 阶段性结论高亮

5. **增强 ConsolePanel**
   - 阶段过滤标签
   - 复制/保存功能
   - 搜索过程控制

### 状态管理扩展

扩展 researchStore：
- `planningData`: 规划阶段数据
- `searchHistory`: 搜索历史（按轮次）
- `analysisSteps`: 分析步骤记录
- `currentPhase`: 当前阶段

### 交互功能

1. 实时更新 (SSE)
2. 折叠/展开 (Accordion)
3. 复制功能 (单条/全部)
4. 保存导出 (JSON/Markdown)
5. 阶段导航 (快速跳转)

### 技术栈

- React + TypeScript + Tailwind CSS
- shadcn/ui 组件 (Accordion, Tabs, ScrollArea, Badge, Card)
- Zustand 状态管理
- SSE 实时通信

### 实现步骤

1. 创建 ProcessVisualizer 主组件
2. 实现 PlanningPanel、SearchPanel、AnalysisPanel
3. 扩展 researchStore 状态
4. 增强 ConsolePanel 功能
5. 更新 App.tsx 布局为三栏式
6. 添加复制/保存功能
7. 测试和优化

这个方案充分利用现有架构，通过添加中央可视化面板实现完整的实时过程展示。
