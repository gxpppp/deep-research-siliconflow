## 支持的模型提供商

1. **SiliconFlow** (默认) - https://api.siliconflow.cn/v1
2. **OpenAI** - https://api.openai.com/v1
3. **Azure OpenAI** - 自定义 endpoint
4. **Anthropic** - https://api.anthropic.com/v1
5. **Google Gemini** - https://generativelanguage.googleapis.com
6. **自定义** - 用户输入任意 OpenAI 兼容的 API 地址

## 数据流

```
用户选择提供商
    ↓
更新 apiBaseUrl (根据提供商预设或自定义)
    ↓
用户输入 API Key
    ↓
获取模型列表 (调用 /api/models 或提供商 API)
    ↓
用户选择/输入模型
    ↓
保存到历史模型列表
    ↓
配置其他参数
    ↓
保存到 settingsStore
```
